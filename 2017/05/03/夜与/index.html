夜与

发表于 2017-05-03 更新于 2023-03-09

焙新酒，点孤灯。

对花语，问东风。

蜂蝶寐，月半弓。

云遮影，树挽藤。

举杯对，我与卿。

醉姮娥，话三更。

语未断，杯莫停。

天下事，谈笑中。

酌复酌，语不清。

灯花落，听鼾声。
