du 命令的使用方法

发表于 2019-04-22 更新于 2023-03-16

我们经常需要在 linux 和 unix 上使用 du 命令来查看文件和文件夹的大小，在这里，我列举一下常用的一些用法。

基本命令参数

-h 以人类可读的方式显示（比如 K, M 和 G）

-g 以 Gb 为单位显示

-m 以 Mb 为单位显示

-k 以 Kb 为单位显示

-d depth 显示指定层级数

-s 只显示一条（相当于 -d 0）
